# E-Commerce-Sales-Analysis
i. Project Objective
This project analyzes customer purchase behavior.
The goal is to identify what type of product is puchase most from which region at what price.

ii.Dataset
Sample Superstore Dataset(kaggle)
Records:- 10,000
Geography:- United States

iii.Tools & Technologies
Python(Numpy,Pandas,Matplotlib) for Data Extraction,Data Cleaning and preparation
Power BI Desktop
DAX (Measures & KPIs)

iv. Key KPIs
Total Sales
Total Profit
Total Orders
Profit Margin

v.Key Insights
a. Total Sales and Profit vs Months
b. Total Sales and Profit By Region
c.Total Sales By Category
d.Sum of Quantity by State(Top 10)

vi. Dashboard Feature
a.Sales and profit Trend over Months
b.Total Orders
c.Sum of Quantity by State

vii.How To Use
Download the .pbix file
Open in Power BI Desktop
Refresh data if needed
